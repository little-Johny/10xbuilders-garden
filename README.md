# 10X Builders: Track Code

Repositorio personal de apuntes, conceptos, entregables y herramientas del curso **10X Builders: Track Code** en [LAB10](https://app.lab10.ai).

---

## Estructura

```
/
├── docs/              # Meta: cómo generar apuntes (prompt, plantilla)
├── week-XX/           # Apuntes por semana del curso (Markdown)
├── ia-tools/          # Plantillas y artefactos transversales (p. ej. briefs)
├── projects/          # Código de práctica y laboratorios (repos anidados)
├── README.md
└── .gitignore
```

### Convención de apuntes

En `week-XX/`, los archivos siguen `NN-nombre-descriptivo.md` (número de lección en la semana + slug). El detalle está en [`docs/LESSON-PROMPT.md`](docs/LESSON-PROMPT.md).

---

## Progreso

| Semana | Carpeta    | Estado (notas) |
|--------|------------|----------------|
| 1      | `week-01/` | Lecciones 1–6 + reflexión |
| 2      | `week-02/` | Lecciones 1–3 (según avance en Lab10) |
| 3+     | —          | — |

*Actualizá la columna de estado a medida que cierres módulos en Lab10.*
